# sop-element-theme

SOP 系统自定义 ElementUI 样式。仅提供 SCSS 文件及相关资源；使用 `storybook` 做控件样式预览

目前依赖版本： `ElementUI` 2.13.2

## 本地预览

```
npm run storybook:serve
```

## 打包为静态文件

```
npm run storybook:build
```
